# @mainsquare/excalidraw-common

## Install

```bash
npm install @mainsquare/excalidraw-common
```

If you prefer Yarn over npm, use this command to install the Excalidraw utils package:

```bash
yarn add @mainsquare/excalidraw-common
```

With PNPM, similarly install the package with this command:

```bash
pnpm add @mainsquare/excalidraw-common
```
